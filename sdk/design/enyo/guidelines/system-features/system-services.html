System Services

This table summarizes the system services available to your app. For further information, please refer to the webOS Service API documentation in the SDK.

Service Brief Description
Accelerometer Captures orientation events and accelerometer data.
Activity Manager Schedules and prioritizes device activities--e.g., apps, services, tasks, network flows.
Alarms Sets a timer to activate on the device either after a specified interval or at a specified date and time.
Application Manager Invokes default handlers for the common resource types or basic device operations.
Bluetooth Uses the Bluetooth Serial Port Protocol (SPP) to access certain Bluetooth devices.
Connection Manager Gets connection status and subscribes to notifications of connection status changes.
db8 Interfaces to an embedded JSON database.
Display Manager Gets events related to the status of the display.
Download Manager Uploads and downloads files over HTTP.
Firewall Opens a port on the firewall.
GPS Gets the current location coordinates and registers for continuous updates.
In-App Payment Allows apps to embed a store directly in their app and process payments for digital content.
Key Manager Provides key management and cryptographic functionality.
Key Service Gets keypress events from headset and volume buttons.
Media Permissions Grants read-only access to media files stored in db8.
Power Management Enters Sleep mode after period of inactivity.
Print Manager Provides webOS printing services.
System Properties Gets the named system properties, including device ID.
System Services Accesses various system settings, including system time.
System Sounds Plays audio feedback in response to user interaction. The sounds play when the message is played, with low latency.
Touch2Share Allows devices to share data at a tap.
Zeroconf Automatically discovers printers, computer services, and other local network device services.
