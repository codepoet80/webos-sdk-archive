Device Shell

The webOS SDK comes with utility programs for logging into the device or Emulator:

  • On Windows, use the PuTTY program - a terminal emulator application which acts as a client for the SSH protocol.

  • On the Mac or Linux, use novaterm or the SSH program.

Once you have a device shell, you can use most of the common Linux shell commands, i.e., cd, cat, ls, cp, du, less, more, and vi. Note that the vi implementation is a version from Busybox, so many of the advanced commands do not work.

Before You Begin

  • You need to download the SDK/PDK and have it installed.

The following applies to a HP webOS device, not the Emulator.

  • Your device has to have developer mode enabled before you can use these utilities.

  • Before you can use PuTTY or SSH, you need to open a command-line prompt while the device is connected and run: 
    pdk-device-install install

    Wait for the device to reboot. This script installs a SSH daemon on your device that you can use to get shell access and that sample scripts use to copy files to the device. The daemon listens to network connections coming over the USB cable. A tcprelay service installed on your desktop system redirects traffic from your localhost to the device.

    Note: If you want to use scp to transfer files with novaterm, you should also do this step. scp is a file transfer over SSH tool.

Device Login

Note that your device needs to be USB-connected and the Emulator not running.

Step 1. Open a command prompt.

  1. On Windows run: 
putty -P 10022 root@localhost
  1. On the Mac OS X or Linux: 
ssh -p 10022 root@localhost

Step 2. Press "Enter" at the password prompt.


Emulator Login

Step 1. Open a command prompt.

  1. On Windows run: 
putty -P 5522 root@localhost
  1. On the Mac OS X or Linux: 
ssh -p 5522 root@localhost

Step 2. Press "Enter" at the password prompt.

Note: To eliminate the password prompt when using SSH, you can run the following at a command-prompt: 

pdk-ssh-init

Using novacom or novaterm

  • On Windows, you can also use novacom to open a shell: 

    novacom open tty://

    Note, however, that novacom on Windows does not give true terminal emulation, so "vi", "less", and tab completion of commands will not work very well.

  • On the Mac OS X or Linux you can use novaterm for both a device and the Emulator: 

    novaterm
    And no login or password is necessary. As noted before, you cannot use scp to transfer files with novaterm unless you run pdk-device-install install.
