JavaScript Library APIs

This section describes loadable JavaScript libraries that can be used in webOS apps and services.

Library Description
Foundations JavaScript utilites for both services and apps. Utilities include those for futures, map reduce, Ajax calls, webOs service calls, array handling, assert operations, string handling, object operations and more.
Media Capture Functionality for recording audio and video and taking camera stills.
Metascene.Videos This library enables you to push the webOS video player or editor onto your application's scene stack.
