Documentation

Welcome to the webOS SDK! The SDK documentation is arranged in three main sections:

  • The Developer Guide provides information on all aspects of application development in webOS, including extensive coverage of the new Enyo app framework. Also covered is the Mojo app framework, PDK apps and plug-ins, Synergy connectors, JavaScript services, in-app payment, and webOS SDK tools. It's a good place to start for most developers.

  • The Design Guide offers guidance to those making user interface decisions for their apps.

  • The Developer Reference documents the wide variety of APIs available in webOS. APIs covered include those for Enyo, Mojo, webOS applications, webOS services, and JavaScript libraries. Also in this section are references for webOS data types and application configuration files.
