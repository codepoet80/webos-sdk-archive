webOS Data Types

This section describes various data types used by webOS applications and services.

Data Types Description
Account Manager Data types for the Account Manager service that Synergy Connector apps interact with.
Activity Manager Data types for the Activity Manager service that prioritizes different tasks for foreground and background processes.
Calendar Data types for calendar data stored in db8 storage.
Contacts Data types for contacts data stored in db8 storage.
db8 Data types used in making API calls for the db8 service that interfaces to an embedded JSON database.
In-App Payment The Receipt object for the in-App Payment service. This service allows apps to embed a store for selling digital content.
Media Indexer Schemas that detail the layout of media JSON data objects.
