PDL_LoadOGL

Description

Dynamically opens a specific OpenGL version.

Note: This API has been available since webOS 3.0.2 (PDK version 303).

Syntax 

PDL_Err PDL_LoadOGL(PDL_OGLVersion version);
Argument Description
version PDL_OGLVersion define: PDL_OGL_1_1 or PDL_OGL_2_0.

Returns 

PDL_INVALIDINPUT - Unknown OGL version sent to PDL_LoadOpenGL.
PDL_NOERROR - Success.
