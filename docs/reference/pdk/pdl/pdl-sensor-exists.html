PDL_SensorExists

Description

Returns flag indicating if sensor exists on this device.

Syntax 

PDL_bool PDL_SensorExists(PDL_SensorType sensor);
Argument Description
sensor Type of sensor to check for - see defines for PDL_SensorType.

Returns 

PDL_TRUE - Sensor exists.
PDL_FALSE - Sensor does not exist or sensor type passed is unknown.

Note: In all cases where an error is returned, your app can call PDL_GetError to get a plain-text error description. This is usually more informative than the returned error code.
