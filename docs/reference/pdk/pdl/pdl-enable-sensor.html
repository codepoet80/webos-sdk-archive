PDL_EnableSensor

Description

Enable or disable a sensor.

Syntax 

PDL_Err PDL_EnableSensor(PDL_SensorType sensor, PDL_bool bEnable)

Both fields are required.

Argument Description
sensor Type of sensor to enable - see defines for PDL_SensorType
bEnable PDL_TRUE (enable) or PDL_FALSE (disable)

Returns 

PDL_INVALIDINPUT - Unknown sensor.
PDL_NOERROR - Sensor is enabled/disabled or sensor is already in the requested state.
PDL_EOTHER - Unable to enable sensor.

Note: In all cases where an error is returned, your app can call PDL_GetError to get a plain-text error description. This is usually more informative than the returned error code.
