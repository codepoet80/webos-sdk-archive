PDL_DismissFullscreen

Description

Returns the plug-in from fullscreen mode back to being a component on a web page.

Your app can call this API any time after it has called PDL_SetFullscreen. It will result in a SDL_VIDEORESIZE event to be added to your event queue. You need to handle this resize event as you would any other resize event. Note that you cannot draw to the plug-in until you have called SDL_SetVideoMode in response to the resize event.

Syntax 

PDL_Err PDL_DismissFullscreen(void);

Returns 

PDL_ESTATE - App is not fullscreen or was not launched as a plug-in.
PDL_EOTHER - Failed to return to plug-in mode.
PDL_NOERROR - Success.

Note: In all cases where an error is returned, your app can call PDL_GetError to get a plain-text error description. This is usually more informative than the returned error code.
