webOS Application APIs

This section documents APIs for interacting with webOS applications, typically via the Application Manager service.

Application Description
Add Contact Launches the Contacts app to enter a new contact.
Browser Loads and views the target specified by a URL.
Calendar Launches the Calendar app to create or access data.
Camera Launches the Camera app to take a picture.
Document Viewers Launches the DocViewer app to browse and view common document file types.
Email Launches the Email app to compose an email, optionally pre-populating the email's contents.
Maps Launches the Maps app to display a map based on the passed-in options.
Messaging Launches the Messaging app to send an IM/SMS/MMS, optionally pre-populating the message's contents.
People Picker A cross-application API provided by the webOS Contacts app for reading contact data one record at a time.
Phone Launches the Phone app to make a call, optionally pre-populating the phone number to call.
Photos Launches the Photos app to view an image.
View File Views a file and launches its associated app.
